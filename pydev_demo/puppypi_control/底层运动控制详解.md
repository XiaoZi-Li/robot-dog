## 1. 系统架构总览

```
 ROS2 决策层 (puppy_ws)
      |
      | /puppy_action (ROS2 Topic)
      |
 ros_udp_bridge
      |
      | UDP:5005 (JSON 指令)
      |
 sit.py ← 运动中枢（右脑）
      |
      | puppy.move() / puppy.gait_config() / runActionGroup()
      |
 HiwonderPuppy（步态引擎）
      |
 servo_controller.py → setServoPulse()
      |
 pwm_servo_control.py → PWMServoControl.setPulse()
      |
 ros_robot_controller_sdk.py → Board.pwm_servo_set_position()
      |
 串口 /dev/ttyS1 (1Mbps)
      |
 STM32 MCU ← PWM信号
      |
 8个 PWM舵机（四足×2个关节）
```

**强化学习关注的两个接入点**：
- **接入点A（高层）**：向 UDP 5005 端口发 JSON，控制行走/转向/坐下等
- **接入点B（底层）**：直接调用 `puppy.move(x, y, yaw_rate)` 或 `setServoPulse(id, pulse, use_time)`

## 3. 通信协议层

### 文件：`ros_robot_controller_sdk.py`

**通信参数**：
- 串口：`/dev/ttyS1`，波特率 **1,000,000 bps**
- 帧格式：`0xAA 0x55 Function Length Data CRC8`

**数据包功能码（PacketFunction）**：

| 功能码 | 值 | 说明 |
|-------|----|------|
| PACKET_FUNC_LED | 1 | LED 控制 |
| PACKET_FUNC_BUZZER | 2 | 蜂鸣器 |
| PACKET_FUNC_MOTOR | 3 | 电机控制 |
| PACKET_FUNC_PWM_SERVO | 4 | **PWM 舵机控制** ← 最常用 |
| PACKET_FUNC_BUS_SERVO | 5 | 总线舵机控制 |
| PACKET_FUNC_IMU | 7 | IMU 数据读取 |
| PACKET_FUNC_RGB | 11 | RGB LED |

**核心方法**：

```python
# 设置 PWM 舵机位置（最关键方法）
board.pwm_servo_set_position(duration, positions)
# duration: 运动时长（秒），如 0.5 表示 500ms
# positions: [(servo_id, pulse), ...]
# 例：board.pwm_servo_set_position(0.5, [[1, 1500], [2, 1200]])

# 读取 PWM 舵机当前位置
board.pwm_servo_read_position(servo_id)  # 返回脉宽值（500~2500）

# 读取 IMU（ax, ay, az, gx, gy, gz）
board.get_imu()  # 返回 6 个 float 值

# 读取电池电压
board.get_battery()  # 返回毫伏值
```

**数据帧格式详解（PWM舵机控制帧）**：
```
0xAA  0x55  0x04  Length  0x01  DurationL  DurationH  Count  [ServoID  PulseL  PulseH]...  CRC8
```
其中 Duration = 运动时长（毫秒，小端序）

## 4. PWM舵机控制层

### 文件：`pwm_servo_control.py`

**类：`PWMServoControl`**

```python
psc = PWMServoControl(board)

# 核心方法
psc.setPulse(servo_id, pulse, use_time)
# servo_id: 1~11（四足用1~8）
# pulse: 500~2500（中位1500，对应约90°）
# use_time: 运动时间（毫秒）

psc.setThreshold(servo_id, min, max)   # 设置舵机行程保护
psc.setDeviation(servo_id, d)          # 设置偏差（-100~100）
psc.saveDeviation()                    # 保存偏差到 servo_deviation.json
```

**舵机 ID 分配（四足8DOF）**：

| 舵机ID | 位置 | 行程限制 |
|--------|------|---------|
| 1 | 前右髋 | 500~2500 |
| 2 | 前右膝 | 650~2300 |
| 3 | 前左髋 | 500~2500 |
| 4 | 前左膝 | 700~2350 |
| 5 | 后右髋 | 500~2500 |
| 6 | 后右膝 | 650~2300 |
| 7 | 后左髋 | 500~2500 |
| 8 | 后左膝 | 700~2350 |

> **注**：舵机2、6（前后右膝）行程 650~2300；舵机4、8（前后左膝）行程 700~2350，保护机械结构。

**偏差文件**：`/app/pydev_demo/puppypi_control/servo_deviation.json`
- 格式：`{"1": 0, "2": 5, "3": -3, ...}`
- 启动时自动加载，确保各舵机在1500时机体水平

## 5. 高层运动 API

### 类：`HiwonderPuppy`（第三方闭源库）

位置：`puppy_env/` 下的 Python 包（编译好的 .so 文件）

**初始化**：
```python
from HiwonderPuppy import HiwonderPuppy, PWMServoParams
puppy = HiwonderPuppy(
    setServoPulse=setServoPulse,  # 注入底层控制函数
    servoParams=PWMServoParams(),  # 舵机参数配置
    dof='8'                        # 8自由度（四足×2关节）
)
```

**步态配置**：
```python
puppy.gait_config(
    overlap_time=0.2,    # 双支撑时间（s），越大步态越稳
    swing_time=0.3,      # 摆腿时间（s），越大步频越低
    clearance_time=0.0,  # 腾空时间（s）
    z_clearance=5        # 抬腿高度（mm）
)
```

**姿态配置（站立基础坐标）**：
```python
puppy.stance_config(
    foot_locations,  # 3×4 矩阵（x/y/z，单位cm），4条腿足端坐标
    pitch=0,         # 俯仰角（rad）
    roll=0           # 横滚角（rad）
)
# 典型站立：foot_locations = [
#     [-0.5, -0.5, -0.5, -0.5],  # x（前后）
#     [0, 0, 0, 0],               # y（左右）
#     [-10, -10, -10, -10]        # z（高度，负值向下）
# ]
```

**运动控制**：
```python
puppy.start()          # 启动步态引擎（必须先调用）
puppy.move(
    x=6.0,             # 前进速度（cm/s），正=前进，负=后退
    y=0.0,             # 侧移速度（cm/s），正=左移
    yaw_rate=0.22      # 转向角速度（rad/s），正=左转
)
puppy.move_stop(servo_run_time=300)  # 停止步态，300ms内稳住
```

**逆运动学（直接控制足端坐标）**：
```python
# 4条腿足端相对坐标控制（3×4矩阵，单位m）
joint_angles = puppy.fourLegsRelativeCoordControl(foot_locations_m)
puppy.sendServoAngle(joint_angles, duration_ms)
```

**其他工具方法**：
```python
puppy.servo_force_run()  # 强制上扭矩（从软件掉电状态恢复）
```

## 6. 动作组执行器

### 文件：`action_group_control.py`

**动作组文件格式（.d6ac）**：
- 本质是 SQLite 数据库
- 表结构：`ActionGroup(Index, Time, C1~C12)`
  - `Time`：该帧持续毫秒数
  - `C1~C12`：12个足端坐标值（float，单位cm）
  - 也支持 `.d6a` 格式（舵机脉宽，int类型，`Servo1~Servo11`）

**核心接口**：
```python
from action_group_control import runActionGroup, stopActionGroup

runActionGroup('sit.d6ac')          # 异步执行（非阻塞）
runActionGroup('stand.d6ac', wait=True)  # 同步等待（最多30s）
stopActionGroup()                   # 立即中断
```

**执行流程**：
1. 读取 SQLite 中每一帧
2. 若数据为 `float` → 调用 `puppy.fourLegsRelativeCoordControl()` 足端IK
3. 若数据为 `int` → 直接 `setServoPulse()` 发脉宽
4. 按 `Time` 字段 sleep 对应毫秒

## 7. UDP 指令接收器

### 文件：`sit.py`（当前运动中枢）

**监听配置**：
- IP：`127.0.0.1`，端口：**5005**
- 协议：UDP，UTF-8 编码

**指令协议（两种格式均支持）**：

```json
# 新协议（推荐）：JSON格式
{"action": "walk", "source": "follow", "extra": "..."}

# 旧协议：纯字符串
walk
```

**支持的动作指令**：

| 指令 | 效果 | 参数 |
|------|------|------|
| `walk` | 前进 | x=6.0 cm/s |
| `turn_left` | 左转 | x=2.5, yaw=0.22 rad/s |
| `turn_right` | 右转 | x=2.5, yaw=-0.22 rad/s |
| `stop` | 停止 | 原地稳住 |
| `sit` | 坐下 | 执行 `sit.d6ac` |
| `stand` | 站立 | 恢复站立步态 |
| `follow_on` | 开跟随模式 | 状态标志 |
| `follow_off` | 关跟随停车 | 状态标志 + stop |

**平滑控制机制**：
- 后台线程每 50ms 更新一次运动量（低通斜坡滤波）
- 前进速度斜坡：`10.0 cm/s²`
- 转向斜坡：`0.9 rad/s²`
- 防止硬切导致机体不稳

## 9. 强化学习接入指南

### 方案A：UDP接口（最简单，高层）

直接向 `127.0.0.1:5005` 发 UDP 指令，不需要修改任何底层代码：

```python
import socket, json

sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
SERVER = ("127.0.0.1", 5005)

def send_action(action: str, source: str = "rl_agent"):
    msg = json.dumps({"action": action, "source": source})
    sock.sendto(msg.encode('utf-8'), SERVER)

# 示例：强化学习控制循环
send_action("walk")          # 前进
send_action("turn_left")     # 左转
send_action("stop")          # 停止
send_action("sit")           # 坐下（用于回合结束重置）
```

**优点**：解耦彻底，不影响其他模块
**缺点**：动作粒度粗，只有6个离散动作，无连续控制量

### 方案B：直接调用 puppy.move()（连续动作空间）

绕过 `sit.py`，直接控制步态引擎，适合 **连续动作空间** 强化学习：

```python
# 配置步态
puppy.gait_config(overlap_time=0.2, swing_time=0.3, clearance_time=0.0, z_clearance=5)

# 站立基础姿态
stance = np.array([
    [-0.5, -0.5, -0.5, -0.5],  # x
    [0.0,  0.0,  0.0,  0.0],   # y
    [-0.10,-0.10,-0.10,-0.10]   # z（单位：m）

puppy.stance_config(stance, pitch=0, roll=0)
puppy.start()

# RL控制循环（连续动作）
def rl_step(action: np.ndarray):
    """
    action: [vx, vy, vyaw]
    vx: [-15, 15] cm/s  前后
    vy: [-5, 5]   cm/s  侧移
    vyaw: [-0.5, 0.5] rad/s 转向

board.enable_reception()

    imu = board.get_imu()  # (ax, ay, az, gx, gy, gz)
    return np.array(imu)
```

### 方案D：舵机脉宽直接控制（最底层）

适合做 **关节级** 强化学习，但需要自己处理逆运动学：

```python
from servo_controller import setServoPulse

# servo_id: 1~8（四足）
# pulse: 500~2500（中位1500，范围约覆盖-60°~+60°）
# use_time: 运动时间（ms）

setServoPulse(servo_id=1, pulse=1500, use_time=200)  # 前右髋，中位，200ms
setServoPulse(servo_id=2, pulse=1800, use_time=200)  # 前右膝，弯曲，200ms

# 批量设置所有腿（需要自己计算每个关节角度）
def set_all_joints(pulses: list, use_time: int = 100):
    """pulses: 长度8的列表，对应舵机1~8"""
    for i, pulse in enumerate(pulses):
        setServoPulse(i + 1, pulse, use_time)
```

**脉宽与角度换算**（近似）：
```
pulse = 1500 + angle_deg * (1000 / 90)
# 1500 → 0°（中位）
# 2500 → +90°
# 500  → -90°
```

### 强化学习推荐框架

```python
import gym
import numpy as np

class PuppyPiEnv(gym.Env):
    """PuppyPi 强化学习环境骨架"""
    
    def __init__(self):
        # 动作空间：vx, vy, vyaw 连续控制
        self.action_space = gym.spaces.Box(
            low=np.array([-15, -5, -0.5]),
            high=np.array([15,  5,  0.5]),
            dtype=np.float32
        )
        # 观测空间：IMU 6轴 + （可扩展：关节角度/视觉特征）
        self.observation_space = gym.spaces.Box(
            low=-np.inf, high=np.inf, shape=(6,), dtype=np.float32
        )
        # 初始化 puppy
        from servo_controller import setServoPulse
        from HiwonderPuppy import HiwonderPuppy, PWMServoParams
        self.puppy = HiwonderPuppy(setServoPulse=setServoPulse, 
                                   servoParams=PWMServoParams(), dof='8')
    
    def reset(self):
        # 重置为站立状态
        from action_group_control import runActionGroup
        runActionGroup('stand.d6ac', wait=True)
        return self._get_obs()
    
    def step(self, action):
        vx, vy, vyaw = action
        self.puppy.move(x=float(vx), y=float(vy), yaw_rate=float(vyaw))
        
        obs = self._get_obs()
        reward = self._compute_reward(obs)
        done = self._is_done(obs)
        return obs, reward, done, {}
    
    def _get_obs(self):
        from ros_robot_controller_sdk import Board
        imu = self.board.get_imu()
        return np.array(imu, dtype=np.float32)
    
    def _compute_reward(self, obs):
        ax, ay, az, gx, gy, gz = obs
        # 示例奖励：保持机体水平（竖轴加速度接近9.8）
        return -abs(az - 9.8) - 0.1 * (abs(ax) + abs(ay))
    
    def _is_done(self, obs):
        ax, ay, az, gx, gy, gz = obs
        return abs(az) < 5.0  # 倒地检测
```

## 10. 关键参数速查

### 步态引擎参数建议范围

| 参数 | 默认值 | 最小 | 最大 | 说明 |
|------|-------|------|------|------|
| `overlap_time` | 0.2s | 0.1 | 0.4 | 双支撑时间，增大→更稳 |
| `swing_time` | 0.3s | 0.15 | 0.5 | 摆腿时间，减小→步频高 |
| `z_clearance` | 5mm | 3 | 15 | 抬腿高度，增大→跨障碍 |
| `x（vx）` | 6 cm/s | -15 | 15 | 前进速度 |
| `yaw_rate` | 0.22 r/s | -0.5 | 0.5 | 转向角速度 |

### 机体坐标系说明

```
前(+x) ↑
左(+y) ←    机体中心
高(+z) ↑

腿序：FR（前右）FL（前左）BR（后右）BL（后左）
列索引：         0         1         2         3
```

### 站立高度与稳定性

| z值(cm) | 说明 |
|---------|------|
| -8 | 低姿（稳定，爬行感） |
| **-10** | **默认站立高度（推荐）** |
| -12 | 高姿（灵活，重心高） |
| -15 | 最高（不稳定） |
